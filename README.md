## Informacje i komunikaty militarne (naniesione na mapę ) dotyczące działań wojennych, podawane zarówno przez wojska i media Ukrainy jak i Rosji.
## Jeśli ostrzał wystąpił w tym samym miejscu, nie nanosiłem go ponownie. 
### Kolor czerowny - Rosja
### Kolor żółty - Ukraina

Dane pochodzą z różnych stron internetowych:
- jeśli prowadzony był ostrzał jakiegoś miasta, oznaczałem punt na mapie kolorem strony, która dokonała ostrzału;
- jeśli któraś ze stron zajęła/odbiła teren, oznaczałem je zgodnie z kolorystyką;
- jeśli strona rosyjska wycofywała się z terenu (obręb Kijowa w kwietniu) to usuwałem kolor czerwony. 

Mapa rozpoczyna się od dnia 1, każda kolejna mapa - to kolejny dzień.
